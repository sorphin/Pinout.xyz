# Uputronics LoRa Zero Expansion Board

Fitted with a HopeRF LoRa module in 434,868 or 915MHz frequency.

Switchable between CE0 or CE1

## Features

* Two LEDs for data and other indications. 
* Robust SMA connector for antennas. 
* Stackable with other Uputronics Boards 
* Open source code and hardware
